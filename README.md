# xExchange

Connect xExchange to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/xexchange) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-xexchange@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-xexchange --url https://cortex.joai.ai/mcp/apps/xexchange
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/xexchange.git ~/.cursor/plugins/local/joai-xexchange
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-xexchange&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWkteGV4Y2hhbmdlIjp7InVybCI6Imh0dHBzOi8vY29ydGV4LmpvYWkuYWkvbWNwL2FwcHMveGV4Y2hhbmdlIn19fQ==) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/xexchange
```
